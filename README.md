# uci-parser-ts

A [UCI](https://en.wikipedia.org/wiki/Universal_Chess_Interface) parser written in TypeScript.

## install

npm

```bash
npm i uci-parser-ts
```

yarn

```bash
yarn add uci-parser-ts
```

pnpm

```bash
pnpm i uci-parser-ts
```
